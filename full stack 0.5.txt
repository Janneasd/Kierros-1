

Selain hakee palvelimelta sivun sisällön HTTP GET -pyynnöllä.
Sitten selain hakee sivun tyylit main.css-tiedostosta sekä JavaScript-koodia sisältävän tiedoston main.js.

selain->palvelin: HTTP GET https://studies.cs.helsinki.fi/exampleapp/spa
palvelin-->selain: HTML-koodi
selain->palvelin: HTTP GET https://studies.cs.helsinki.fi/exampleapp/main.css
palvelin-->selain: main.css
selain->palvelin: HTTP GET https://studies.cs.helsinki.fi/exampleapp/spa.js
palvelin-->selain: spa.js

Selain alkaa suorittaa js-koodia joka pyytää JSON-datan palvelimelta

selain->palvelin: HTTP GET https://studies.cs.helsinki.fi/exampleapp/data.json
palvelin->selain: [{content: "spa", date: "2022-01-12T11:29:43.092Z"},…]

Datan saapuessa selain suorittaa tapahtumankäsittelijän, joka renderöi muistiinpanot single page apin ruudulle.
